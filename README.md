# SmartWeather

Welcome to **SmartWeather**, an intuitive weather forecasting application powered by AI to give you personalized weather insights and notifications tailored just for you.

---

## Overview
SmartWeather combines real-time weather data with advanced AI models to provide accurate, localized forecasts and actionable insights. Whether you're planning your day or preparing for unexpected weather, SmartWeather keeps you informed.

## Features
- Real-time weather updates for any location worldwide
- AI-driven personalized weather insights and alerts
- Hourly, daily, and weekly forecast views
- Interactive weather maps and radar
- Customizable notifications for severe weather and changes

## Tech Stack
- **Frontend:** React.js with Tailwind CSS
- **Backend:** Node.js, Express.js
- **API:** OpenWeatherMap API
- **AI:** TensorFlow.js for predictive modeling
- **Deployment:** Docker, hosted on AWS

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/n0ah95/SmartWeather.git
   ```
2. Navigate to the project directory:
   ```
   cd SmartWeather
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Create a `.env` file with your OpenWeatherMap API key:
   ```
   REACT_APP_WEATHER_API_KEY=your_api_key_here
   ```
5. Start the development server:
   ```
   npm start
   ```

## Usage
- Open your browser at `http://localhost:3000`
- Search for a location using the search bar
- View detailed weather forecasts and AI-driven insights
- Customize your notification preferences in the settings page
